jQuery

¿Qué es y para qué usar JQuery?

jQuery es una popular biblioteca de JavaScript. Fue creada por John Resig en el 2006 con el objetivo de facilitarle el uso de JavaScript en los sitios web a los desarrolladores. jQuery comprime varias líneas de código en una sola función, de forma que no es necesario volver a escribir bloques enteros de código para lograr una sola tarea
